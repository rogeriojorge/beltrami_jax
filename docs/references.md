## Relation to this repository

These references serve different purposes here:

- the SPEC papers and manual define the discrete solve this repository is reproducing
- the Taylor and MRxMHD papers explain the physical role of Beltrami states
- the Taylor-state solver literature provides context for alternative numerical formulations and future comparisons
