SPECTRE compare-case fixtures
=============================

These fixtures are compact validation assets copied from the public SPECTRE
repository and from fresh local SPECTRE coefficient exports.

Origin:

- SPECTRE remote: https://gitlab.com/spectre-eq/spectre.git
- assessed SPECTRE commit: 08e358a
- source directory: tests/compare/<case>/

Each case contains:

- input.toml: the public SPECTRE compare input
- reference.h5: the public SPECTRE reference output
- fresh_spectre_export.npz: `spectre.get_vec_pot_flat(test)` output generated by
  `tools/export_spectre_vecpot_npz.py`

These files are not required by ordinary users. They make the SPECTRE
vector-potential parity target reproducible in `beltrami_jax` CI without a local
SPECTRE checkout.
